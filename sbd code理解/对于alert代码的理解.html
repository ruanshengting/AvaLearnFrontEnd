对于alert代码的理解 
        问题：
        1.transform: translate(-300%);和left: 300%;的关系
        2.transition: 0.5s transform ease-in-out;这个是什么
        3.为什么input的class要弄成sbd_radio_group sbd_radio_group_1，直接写sbd_radio_group_1，把sbd_radio_group省略不行吗？？
        4.我这样调用定时器为什么不行
      
        理解：
        整体思路：
        input在html的最顶层，这样可以方便用input作为选择器的开头来控制页面
        CSS方面：
        1.input不见了是因为   li.slide 中的position: absolute;top: 0; 把input radio给盖住了
        2.选择input之后，根据是第几个input设定要移动多少
        js方面：
        通过map,增加了labelFor属性和classIndex,class。classIndex相当于给label标签编号，对应的是input的class:sbd_radio_group_{classIndex},由此可以让input 被选中。这里的class只作为选择器来用，让它可以在js中被选中为checked，并不给他增加样式（input会被盖住）
